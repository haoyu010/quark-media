302 播放服务

已内置在本容器，无需另外部署。播放时本服务返回 302，客户端直连夸克 CDN（不经 NAS 流量）。

Built-in
工作方式
STRM 内容形如 http://飞牛IP:18025/play/{fid}/xxx
Emby 读 STRM → 请求本服务 → HTTP 302 跳到夸克 m3u8 → 播放器直连夸克。
Docker 监听固定 0.0.0.0:18025（compose 映射），一般不用改。
写入 STRM 的前缀
必须是 Emby / 电视 / 手机能访问到的地址。改完后新生成的 STRM 用新前缀；旧 STRM 需重跑任务才会更新。
0 表示只手动触发；大于 0 则定时跑转存→STRM。

STRM 输出

路径不在 Web 配置。Docker 用卷挂载 / 环境变量自定义，Emby 扫该目录入库。

容器内固定 /app/strm。
宿主机改 QM_STRM_HOST（compose 映射）或挂载目录即可。
流程：转存完成 → 生成 .strm → Emby 刷库（不做扫库）。
